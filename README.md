# Password Strength Analyzer

A Python-based tool that evaluates password strength based on:

- Password Length
- Uppercase Letters
- Lowercase Letters
- Numbers
- Special Characters

## Features

- Strength rating (Weak, Moderate, Strong, Very Strong)
- Password improvement suggestions
- Easy-to-use command line interface

## Technologies Used

- Python
- Regular Expressions (re)

## How to Run

```bash
python password_strength_analyzer.py
